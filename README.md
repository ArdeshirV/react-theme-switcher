# react-theme-switcher

A ReactJS-based switch to change style of page from **Light** to **Dark** or versa.
 
![](theme-switcher.gif)

## Features

- CSS Animation
- React JS
- React Context
- React State Management

### Install dependency

```
npm install
```

### Start and serve service

```
npm start
```
